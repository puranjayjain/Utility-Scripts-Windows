# Utility-Scripts-Windows
This contains some useful utility batch files (*.bat) for your windows pc

# Compatibility
The scripts have been tested on and have been found working on the following windows versions
 - Windows 8 (x64)
 - Windows 8.1 (x64)
 - Windows 10 (x64)
